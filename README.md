# GTA VI landing page

Page de présentation statique pour la précommande de GTA VI.

## Aperçu local

Ouvrez simplement [index.html](index.html) dans votre navigateur, ou démarrez un petit serveur local :

```bash
python3 -m http.server 8000
```

Puis ouvrez http://localhost:8000/ dans votre navigateur.

## Déploiement

Cette page est entièrement statique et peut être publiée facilement sur :
- GitHub Pages
- Netlify
- Vercel

Pour GitHub Pages, publiez la branche principale avec [index.html](index.html) à la racine du dépôt.
